#ElGamal 타원곡선
-
임의의 두 원소 P, Q가 적당한 정수 x에 대해 Q=xP를 만족할 때, P와 Q만을 알고 x를 구하는 문제를 말한다.
이때의 x를 구하는 것은 매우 어렵다.

2의 거듭제곱 꼴로 표현되는 소수를 사용하면 field값을 높일 수 있다.
비트코인의 경우 $2^{256} - 2^{32} - 977$을 사용한다.
